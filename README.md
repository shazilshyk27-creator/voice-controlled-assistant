# Voice Controlled Assistant

A Python-based voice-controlled assistant that can understand spoken commands and perform tasks such as opening applications, searching the web, telling the time, and responding using text-to-speech.

## Features

- Speech Recognition
- Voice Responses (Text-to-Speech)
- Open Applications
- Web Search
- Time & Date Commands
- Custom Commands

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- PyAudio
